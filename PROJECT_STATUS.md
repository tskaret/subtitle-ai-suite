```bash

cd subtitle-ai-suite

# 2. Create virtual environment

# 4. Setup environment

# 5. Download AI models

# 6. Test installation

# 7. Run application
python src/main.py
```

### **Docker Installation:**
```bash

docker build -t subtitle-ai-suite .

# Run with GPU support
docker run --gpus all -v ./input:/app/input -v ./output:/app/output subtitle-ai-suite
```
